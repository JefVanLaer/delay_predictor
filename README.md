# delay_predictor
This repo contains a POC for a shipping delay predictor to be used by freight carriers.

# data
The data used in the project is sourced from the following sources:
- The US Marine Cadastering System (MCS) for vessel tracking data. (https://hub.marinecadastre.gov/pages/vesseltraffic)
- The world port index for port information. (https://msi.nga.mil/Publications/WPI)

# code
The code is organized into the following folders:
- notebooks: contains Jupyter notebooks for data exploration and model development.
- src: contains the source code for data processing, feature engineering, and model training.
- data: contains the raw and processed data used in the project.
